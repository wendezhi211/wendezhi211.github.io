图片居中 ​

创建一个具有背景图像的容器，背景图像会被拉伸或收缩以覆盖整个容器，并且会在容器的中心位置显示:

css
.container {
  background-image: url('your-image.jpg');
  background-size: cover;
  background-position: center;
}

或者:

ts
// 选择容器元素
const container = document.querySelector('.top')!;

// 选择容器内的图片元素
const image = document.querySelector<HTMLImageElement>('.top img')!;

// 当容器大小发生变化时执行的函数
const handleResize = () => {
    // 获取容器的宽度和高度
    const containerWidth = container.clientWidth;
    const containerHeight = container.clientHeight;

    // 获取图片的原始宽度和高度
    const imageWidth = image.naturalWidth;
    const imageHeight = image.naturalHeight;

    // 计算容器的宽高比
    const containerAspectRatio = containerWidth / containerHeight;

    // 计算图片的宽高比
    const imageAspectRatio = imageWidth / imageHeight;

    // 根据容器和图片的宽高比比较，设置不同的样式类以填充或包含图片
    if (containerAspectRatio > imageAspectRatio) {
        container.classList.add('fill');
        container.classList.remove('contain');
    } else {
        container.classList.add('contain');
        container.classList.remove('fill');
    }
};

// 创建 ResizeObserver 实例，监测容器大小的变化
const resizeObserver = new ResizeObserver(handleResize);
resizeObserver.observe(container);
// 首次加载图片时
image.onload = () => {
    handleResize()
};
scss
.top {
    position: relative;
    display: flex;
    justify-content: center;
    /* 水平居中 */
    align-items: center;
    /* 垂直居中 */
    overflow: hidden;
    /* 隐藏超出容器的部分 */

    &.fill img {
      width: 100%;
      height: auto;
    }
    &.contain img {
      width: auto;
      height: 100%;
    }
  }

CSS样式定义块 ​

在CSS中，你可以使用自定义属性（CSS变量）来定义一些值，以便在整个样式表中重复使用这些值。这样可以使样式更易于维护和修改。以下是如何定义和使用CSS变量的方法：

定义CSS变量： ​

可以在选择任何CSS规则块内部或者在根选择器 :root 中定义CSS变量。一般来说，:root 伪类用于设置全局的CSS变量，这样它们将在整个文档中可用。

css
:root {
  --primary-color: #3498db;
  --secondary-color: #f39c12;
}

使用CSS变量： ​

可以在样式规则中使用定义的CSS变量，通过 var() 函数引用它们。这样可以使样式更加灵活和易于维护。

css
.button {
  background-color: var(--primary-color);
  color: var(--secondary-color);
}

JavaScript中的CSS变量： ​

还可以使用JavaScript来动态修改CSS变量的值。通过JavaScript，你可以实时更改页面的外观。

javascript
document.documentElement.style.setProperty('--primary-color', '#ff0000');

这样，就可以根据需要定义和使用CSS变量，使样式表更加灵活和易于维护。

变形基点 ​

scss
transform-origin:

transform-origin 是一个CSS属性，用于指定一个元素变换的原点，即变换的中心点。这个属性通常与 transform 属性一起使用，用于控制元素的旋转、缩放、倾斜等变换效果的原点位置。

transform-origin 的可取值包括： ​

  1. 关键字值：

    • top
    • bottom
    • left
    • right
    • center

  2. 百分比值：相对于元素自身的宽度和高度的百分比值，例如 50% 50% 表示元素的中心点。

  3. 长度值：可以使用像素（px）、em、rem 等长度单位来指定原点的位置。

  4. 组合值：可以同时指定水平和垂直方向的原点位置，比如 top left、center center 等。
